# ChoreChamp

A family chore management app that helps caregivers assign chores and rewards to children, while kids earn tokens for completing tasks.

## Tech Stack

- React 19 + TypeScript + Vite
- Tailwind CSS 4 + shadcn/ui
- Convex (backend/database)
- Clerk (authentication)
- React Router v7

## Prerequisites

- Node.js
- Bun

## Getting Started

```bash
bun install
bun run dev
```

## Folder Structure

- `src/pages/` — Application pages
- `src/components/` — Reusable components
- `src/mocks/` — Mock data
- `convex/` — Backend schema and functions
